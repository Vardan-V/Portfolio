Crypt Raider

First-Person Puzzle Game made with Unreal Engine 5

Back to Projects GitHub

Description

"Crypt Raider" is a first-person puzzle game I worked on using C++ and Unreal Engine 5. The objective of this game is for the player to reach the end of the crypt and successfully leave with the golden statue. The player will have to discover secrets and solve puzzles to accomplish this task.

Project Features

  • Trigger and Collision Detection
  • Overlap Events
  • Grabbing & Dropping Game Objects
  • Level Lighting (Skylight, Directional & Spotlight)
  • Modular Level Design


Tools & Assets

  • C++ Programming Language
  • Unreal Engine 5
  • Medieval Dungeon Asset Pack
