Warehouse Wreckage

A First-Person Destruction Game made with Unreal Engine 5

Back to Projects GitHub

Description

"Warehouse Wreckage" is a first-person video game I worked on using Unreal Engine 5. This game was made only using the blueprints feature in Unreal. In this game, the player can shoot heavy projectiles using the "space bar" key. The player has 20 shots and is free to shoot anywhere in the level (preferably at the barrels to showcase the physics). When ammunition is depleted, the level will restart.

Project Features

  • Spawning Projectiles using Player Input
  • Level Design using Geometry Brushes
  • Loading Levels using Delay Timers
  • Materials & Lighting

Tools & Assets

  • Unreal Engine 5
  • Industry Prop Pack 6
