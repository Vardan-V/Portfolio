Toon Tanks

A Third-Person Tank Shooter made with Unreal Engine 5

Back to Projects GitHub

Description

"Toon Tanks" is a third-person video game I worked on using C++ and Unreal Engine 5. In this game, you are a tank with the objective of killing all turrets in the level. You must do this while also avoiding getting hit by turret shells.

Project Features

  • Player Input
  • Mesh Rotation
  • Turret Targeting
  • Particle Effects
  • Sound Effects
  • Camera Movement & Shake

Tools & Assets

  • C++ Programming Language
  • Unreal Engine 5
  • ToonTanksAssetPack (Udemy)
