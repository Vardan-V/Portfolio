Description

"Obstacle Assault" is a third-person video game I worked on using C++ and Unreal Engine 5. It also implements gameplay using Unreal Engines Blueprint feature. The objective of this game is to reach the end location while avoiding obstacles and falling off the map.

Blueprints

Used Unreal Engine's Blueprints to implement:

  • Player Movement
  • UI Display
  • Level Loading
  • Game End Conditions

Level Design & Obstacles

  • Created a "Moving Platform" C++ class that enables obstacles to move in various directions.
  • Designed a level using assets to create engaging gameplay (not too hard, not to easy).
