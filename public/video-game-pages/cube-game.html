Cube Game

A Third-Person Obstacle Avoiding game made with Unity Engine

Back to Projects GitHub

Description

"Cube Game" is a third person video game I created using C# and the Unity Engine. In fact, this was the first video game project that I tackled. In this game, the player controls a red cube with the objective of reaching the end of the level while avoiding obstacles and falling off the map. Upon reaching the end of the level, they will progress to the next. If they fall or hit an obstacle, it will restart the current level.

Project Features

  • Prefab Creation
  • Multi-Level Design
  • UI Menus
  • User Input for Player Control
  • Trigger Events
