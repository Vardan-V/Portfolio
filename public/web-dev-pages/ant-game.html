Ant Game

A Mobile Game made with Java and CodenameOne

Back to Projects GitHub

Description

"Ant Game" is a mobile game I have developed for my CSC 133 class (Object Oriented Programming - Graphics) at Sac State. In this game, you are controlling an Ant with the objective of reaching all flag stations in the correct order. You must do this while avoiding spiders and starvation. The ant will have health, number of lives, a food level and speed. If you collide with a spider, your health will be reduced. You have three lives. If your health or food level reaches zero, you will lose a life and start over. Losing three lives will result in a game over.

UI

Ant game features a UI representing a tablet. It displays Game Objects, Player Information as well as available commands through the use of buttons.

Game Objects:

  • Ant - Solid Pink Circle
  • Spider - Hollow Black Triangle
  • Flags - Solid Blue Triangle
  • Food Station - Solid Green Square

Player Information

  • Time - Game Ticks Elapsed (each tick is 20ms)
  • Lives Left - Starts with 3
  • Last Flag Reached - Current Flag Number Reached
  • Food Level - Current Player Food Level
  • Health Level - Current Player Health
  • Sounds - (Able to toggle between on and off)

Project Features

  • Buttons that allow for gameplay actions such as: Accelerate, Brake, Steer Left/Right
  • Pause & Resume Game
  • Sound Effects & Background Music
  • Bounds Checking for Game Objects
  • Collision Detection & Handling
  • Game Logic - (Win & Lose Conditions)

Programming Languages, Tools & Frameworks

  • Language: Java
  • IDE: Eclipse
  • CodenameOne Framework
  • JavaFX
