Movie Search

Movie Database Using FStreams & Tokenization

Back to Projects GitHub

Description

"Movie-Search" is a text-based project I worked on that simulates a movie database. In this project, I utilize FileStreams that will open a text file named "db.txt". If none exists, then one will be created. The user will then be shown a list of available operations to perform. The program will then tokenize the file contents using delimiters and perform the requested operations.

Project Features

  • Movie Lookup by Criteria
  • Text-Based Menu
  • New Movie Entry
  • Delete Entry
  • Display All Movies
  • Quit Program

Programming Languages & Tools

  • Java
  • Visual Studio Code
  • Notepad (To read and write data)
